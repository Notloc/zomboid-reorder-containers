title=[B42/41] Reorder Containers - Backpack Orders
description=[b]Reorder Containers[/b] is an inventory mod that allows you to customize the order in which containers and backpacks appear in the inventory.
description=
description=Simply drag and drop the container icons to rearrange them to your liking.
description=They'll even remember their preferred order if you drop the backpack and re-equip it later.
description=
description=
description=You also have the option to rearrange containers in the loot window, [b]but you need to turn it on using the options at the bottom of the loot window (next to the lock)[/b]
description=
description=Lastly, you can manually set the sorting priority for containers if you need to. Just select the container and click the options icon at the bottom right of the window (next to the lock)
description=
description=The drag and drop system will automatically handle the sorting priority by default.
description=
